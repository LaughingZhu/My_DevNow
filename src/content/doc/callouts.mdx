## 前言：

为了更好的支持文章内容，从现在开始，DevNow 支持**标注**了。

目前只支持四种标注类型：

- note：关注
- tip：提示
- caution：警告
- danger：危险

## 使用方法

直接在 `.mdx` 文件中，使用 ::: 来表示。

```md
:::type[title]
具体的内容
:::
```

## 效果展示

:::note[关注]
记得关注DevNow，您的支持对我来说非常的重要。
:::

:::tip[提示]
记得关注DevNow，您的支持对我来说非常的重要。
:::

:::caution[警告]
记得关注DevNow，您的支持对我来说非常的重要。
:::

:::danger[危险]
记得关注DevNow，您的支持对我来说非常的重要。
:::
